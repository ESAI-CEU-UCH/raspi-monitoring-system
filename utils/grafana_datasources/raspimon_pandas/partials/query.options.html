Consolidated by
  • Consolidation only is used when number of real points is greater than maxDataPoints
  • Series data is consolidated by computing reduction over uniform-spaced windows
  • By default 'last' consolidation is used
Alias for shorter legend names
  • Every series has a legend name displayed in the graph
  • You can change this name by any other using this parameter
Linear transformation
  • Series data can be transformed linearly
  • A scale transformation (Mul) and a translation (Add) are available
  • By default Mul=1.0 and Add=0.0
Max data points
  • Parameter maxDataPoints is used by every request to raspimon_pandas
  • This parameters limits maximum number of points returned by any query
  • When real values is greater than maxDataPoints, they are consolidated
  • Consolidation operation depends on consolidateBy as indicated in the query
  • Consolidation will effect series legend values
  • High values to maxDataPoints can be harmful to performance
